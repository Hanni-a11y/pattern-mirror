# Pattern Mirror

Pattern Mirror is a reflection-based AI agent designed to help people notice and shift unhelpful thought patterns in everyday situations.

Instead of giving advice, the agent helps users slow down, reflect, and recognize the patterns behind their reactions.

## Purpose

Many of our reactions are shaped by automatic thought patterns. Pattern Mirror helps users become aware of these patterns and gently question them.

The goal is to support:
- self-awareness
- emotional regulation
- reflection before reaction
- personal growth

## How it works

Pattern Mirror guides the user through a simple reflection flow:

1. The user recalls a situation from the day.
2. The agent helps identify the reaction.
3. The user explores what exactly bothered them.
4. The agent highlights a possible thought pattern.
5. The user receives a reflection question and a short micro-practice.

## Example starter questions

- What situation today made you pause?
- Did something today trigger a strong reaction in you?
- Was there something today that bothered or annoyed you?
- Is there a situation you keep replaying in your mind?

## Design principles

Pattern Mirror follows three simple principles:

**Reflection instead of advice**  
The agent asks questions rather than telling users what to do.

**Short interactions**  
Each reflection takes only a few minutes.

**Awareness before change**  
Recognizing a pattern is the first step to changing it.
